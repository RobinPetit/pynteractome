get_lcc_score(genes, nb_sims, shapiro=False, shapiro_threshold=0.05)[source]¶

Get the $z$-score and the empirical p-value of the LCC size of given genes.

PARAMS:
genes (set): gene set nb_sims (int): minimum number of simulations for probability distribution estimation shapiro (bool): True if normality test is needed, False otherwise (default False) shapiro_threshold (float): statistical threshold for normality test
Returns:$(z, p_e, N)$ if shapiro is True and $(z, p_e)$ otherwise; where $z$ is the $z$-score of the LCC size, $p_e$ is the associated empirical $p$-value and $N$ is True if Shapiro-Wilk normality test $p$-value >= shapiro_threshold and False otherwise
RAISE:
ValueError if not enough simulations have been performed
