Representation of \(\Omega_\alpha\) (see pathogenic_topology_analysis for definition).

add(graph, term, genes)[source]¶

Add a 3-uple \((\Xi_\alpha(\sigma), t, \sigma)\) where genes is \(\sigma\), graph is \(\Xi_\alpha(\sigma)\) and term is \(t\).

Note: this tuple being in \(\Omega_\alpha\) (so an IsomorphismCounts object indicates that \((t, \sigma) \in \Lambda_\alpha\), i.e. \(\sigma \in \mathcal P_\alpha(\mu(t))\).

contains(graph)[source]¶

Return True if graph (\(= \Xi_\alpha(\sigma)\)) is contained within the object and False otherwise.
