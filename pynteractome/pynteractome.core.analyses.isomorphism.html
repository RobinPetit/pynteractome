pynteractome.core.analyses.isomorphism.entropy(S)[source]¶

Compute the entropy induced by the isomorphism classes.

Parameters:S (list) – a list of isomorphism classes
Returns:\((H(S), N)\) where \(N\) is the number of isomorphism classes and:
\[H(S) = K(|S|)\sum_{s \in S}\frac {|s|}{|T|}\log_2\frac {|s|}{|T|},\]

where \(T = \bigsqcup_{s \in S}s\) is the set of which S is a partition and \(K(|S|) = \frac 1{\log|S|}\) is the constant normalizing the entropy in \([0, 1]\).

Note the following convention: \(H(\emptyset) := 0\).
Return type:tuple
