pynteractome.core.analyses.isomorphism module¶

pynteractome.core.analyses.isomorphism.entropy(S)[source]¶

Returns $K(|S|)sum_{s in S}frac {|s|}{|T|}log_2frac {|s|}{|T|}$, where $T = bigsqcup_{s in S}s$ is the set of which S is a partition and $K(|S|) = frac 1{log|S|}$ is the constant normalizing the entropy in $[0, 1]$.
