pynteractome.core.analyses.lcc.normality_test_lcc(integrator, p_threshold=0.01)[source]¶

Evaluate whether the LCC size of uniformly sampled disease modules follows a normal distribution.

Parameters:
  • integrator (LayersIntegrator) – the layers integrator
  • p_threshold (float) – the statistical threshold. If \(p < p_{\text{threshold}}\), then the LCC size is considered to not be distributed normally.
Returns:

Mapping HPO term \(\rightarrow (p, n, m)\) where \(p\) is the \(p\)-value of Shapiro-Wilk normality test, \(m\) is the number of genes, and \(n\) is a boolean describing whether the LCC size of uniformly sampled disease modules of size \(m\) are distributed randomly.
Return type:

dict
